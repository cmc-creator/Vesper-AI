# Vesper-AI
My AI Friend

## Getting Started

This is a [Next.js](https://nextjs.org/) project for Vesper AI - your personal AI companion.

### Development

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Deployment on Vercel

This project is optimized for deployment on [Vercel](https://vercel.com).

To deploy:

1. Push your code to GitHub
2. Import your repository on Vercel
3. Vercel will automatically detect it as a Next.js project
4. Click "Deploy"

Vercel will automatically detect the Next.js framework and configure the build settings.
