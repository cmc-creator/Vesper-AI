## Setup Instructions

1. **Install Dependencies**
   ```
   cd C:\Users\conni\VesperApp
   npm install
   ```

2. **Add Your API Key**
   - Copy `.env.example` to `.env`
   - Add your Anthropic API key to the `.env` file:
     ```
     ANTHROPIC_API_KEY=sk-ant-your-key-here
     PORT=3000
     ```

3. **Start Vesper**
   ```
   npm start
   ```

4. **Open Your Browser**
   - Go to: http://localhost:3000
   - Talk to Vesper anytime!
